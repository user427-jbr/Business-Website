Service Overview

I design and implement automated workflows to connect your favorite apps and services. This includes syncing files, sending notifications, collecting data, generating PDF documents, and automating repetitive manual tasks using both Cloud Flows and Desktop Flows (RPA).

Licensing Information

Power Automate has a flexible licensing structure depending on your automation complexity and data sources:

  • Included in M365: Standard cloud flows triggered by standard apps (SharePoint, Outlook, Forms) are typically included for free in most Microsoft 365 plans.
  • Power Automate Premium: For around $15 per user/month, this allows a user to run unlimited flows using Premium Connectors (e.g., HTTP requests, SQL, Dataverse).
  • Power Automate Process: If an automated process operates independently of a specific user (e.g., a central organization-wide data sync), the flow itself can be licensed (approx. $150/bot/month).
  • RPA Add-ons: Utilizing Desktop Flows for unattended Robotic Process Automation (controlling desktop UI without user intervention) requires additional unattended RPA licenses.
