Travel Expenses App

Power Apps | Power Automate | PDF Generation

✈️

Project Overview

A Power Apps solution designed to streamline the travel expense approval process. Employees can easily submit their expenses to their supervisors for approval, and the system automatically generates standardized PDF reports for accounting.

Key Features

  • Submission of travel expenses with receipt attachments
  • Direct supervisor approval workflow
  • Automated PDF report generation via Power Automate
  • Integration with SharePoint for data and file storage
  • Mobile-friendly interface for on-the-go expense tracking
  • Email notifications for status updates

Technologies Used

Power Apps SharePoint Power Automate Microsoft 365

Results

The implementation completely eliminated manual paperwork and accelerated the approval process. Accounting now receives standardized, perfectly formatted PDF reports, drastically reducing processing time and manual errors.
